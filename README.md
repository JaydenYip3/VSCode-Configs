# VSCode-Configs

Simple reference for the VS Code config files in this repository.

## Files:

### `settings.json`
Main VS Code preferences for editor behavior, formatting, Git, and Vim integration.
Includes relative line numbers, highlighted yank, and quick navigation helpers.

### `keybindings.json`
Custom keyboard shortcuts that override VS Code defaults.
Maps terminal and source-control shortcuts and adds pane navigation with `Ctrl+h/j/k/l`.
